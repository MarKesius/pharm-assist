# Pharm Assist Study

Διαδραστική web εφαρμογή επανάληψης για τις εξετάσεις **Τεχνικού Φαρμάκων, Καλλυντικών και Παρεμφερών Προϊόντων**.

Περιλαμβάνει **349 ερωτήσεις** με απαντήσεις, και **16 διαδραστικά κουίζ** (8 για την Ομάδα Α και 8 για την Ομάδα Β).

## Χαρακτηριστικά

### Ερωτήσεις & Απαντήσεις
- Δύο ομάδες: Ομάδα Α (105 ερωτήσεις) και Ομάδα Β (244 ερωτήσεις)
- Accordion UI με εμφάνιση απάντησης με κλικ
- Keyboard accessible πλοήγηση με Enter/Space
- Link προς τον κατάλογο ερωτήσεων PDF

### Κουίζ & Αξιολόγηση
- **16 διαδραστικά κουίζ** - 8 ανά ομάδα
- Επιλογή κουίζ από dropdown menu
- Εμφάνιση ερωτήσεων μία προς μία
- 4 επιλογές απάντησης ανά ερώτηση
- Πλοήγηση μεταξύ ερωτήσεων (προηγούμενη/επόμενη)
- Αποτελέσματα με ποσοστό επιτυχίας
- Εξηγήσεις για τις σωστές απαντήσεις
- Ένδειξη προόδου (progress bar)

### Σχεδίαση
- Responsive εμφάνιση για κινητό και desktop
- Modern UI με gradient backgrounds
- Smooth animations και transitions
- Hamburger menu για κινητά με κουμπιά

## Πώς να χρησιμοποιήσετε

### Ερωτήσεις & Απαντήσεις
1. Πατήστε **Ομάδα Α** ή **Ομάδα Β** στο μενού
2. Κάντε κλικ σε μια ερώτηση για να δείτε την απάντηση
3. Κάντε κλικ ξανά για να κρύψετε την απάντηση

### Κουίζ
1. Πατήστε το **hamburger menu (☰)**
2. Επιλέξτε **Κουίζ**
3. Επιλέξτε το κουίζ που θέλετε από τη λίστα
4. Πατήστε **Έναρξη**
5. Απαντήστε τις ερωτήσεις
6. Δείτε τα αποτελέσματα με αναλυτικές εξηγήσεις

## Εκτέλεση τοπικά

Δεν απαιτείται build step ή server. Άνοιξε απλώς το `index.html` στον browser.

## Deploy

Το project είναι static HTML/CSS/JavaScript, οπότε μπορεί να γίνει deploy σε:
- **Vercel** (Framework Preset: Other)
- **GitHub Pages**
- **Netlify**
- Οποιοδήποτε web server

## Τεχνολογίες

- Vanilla HTML5
- CSS3 (Grid, Flexbox, Animations)
- Vanilla JavaScript (ES6+)
- Google Fonts (Inter)
- JSON για δεδομένα κουίζ

## Σημείωση

Το περιεχόμενο παρέχεται αποκλειστικά ως βοήθημα μελέτης και δεν αντικαθιστά επίσημο εκπαιδευτικό υλικό ή οδηγίες πιστοποιημένων εκπαιδευτών. Το κουίζ περιλαμβάνει εξηγήσεις και λογικές αιτιολογήσεις για καθεμία από τις σωστές απαντήσεις.
