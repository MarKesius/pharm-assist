# Pharm Assist Study

Διαδραστική web εφαρμογή επανάληψης για τις εξετάσεις **Τεχνικού Φαρμάκων, Καλλυντικών και Παρεμφερών Προϊόντων**.

Περιλαμβάνει **349 ερωτήσεις** με απαντήσεις, οργανωμένες σε δύο ομάδες.

## Χαρακτηριστικά

- Δύο ομάδες ερωτήσεων: Ομάδα Α (105 ερωτήσεις) και Ομάδα Β (244 ερωτήσεις)
- Accordion UI με εμφάνιση απάντησης με κλικ
- Responsive εμφάνιση για κινητό και desktop
- Keyboard accessible πλοήγηση με Enter/Space
- Link προς τον κατάλογο ερωτήσεων PDF
- Buy Me a Coffee link για υποστήριξη

## Δομή αρχείων

```text
├── index.html
├── group_b.html
├── omada_a.js
├── omada_b.js
├── style.css
├── README.md
└── 29_TEXNIKOS_FARMAKON_KALLYNTIKON_KAI_PAREMFERON_PROIONTON.pdf
```

## Εκτέλεση τοπικά

Δεν απαιτείται build step ή server. Άνοιξε απλώς το `index.html` στον browser.

## Deploy στο Vercel

Το project είναι static HTML/CSS/JavaScript, οπότε στο Vercel μπορεί να γίνει deploy απευθείας από GitHub με Framework Preset: **Other** και χωρίς build command.
